Deploy to Vercel

Follow these 4 simple steps

Step 1 of 4

📦 Step 1: Install Node.js Dependencies

First, install the required packages locally:

cd C:\Users\kriti\OneDrive\Desktop\guviPS
npm install

This will install Express, MongoDB, Redis, and other dependencies.

🗄️ Step 2: Setup Free Databases

A) PlanetScale (MySQL) - Free Forever

  • Go to planetscale.com
  • Sign up with GitHub
  • Create new database: "user-registration-db"
  • Get connection details
Go to PlanetScale

B) MongoDB Atlas (Free M0)

  • Go to MongoDB Atlas
  • Create FREE M0 cluster (look for "Shared" tier)
  • Allow access from anywhere (0.0.0.0/0)
  • Get connection string
Go to MongoDB

C) Redis Cloud (Free 30MB)

  • Go to Redis Cloud
  • Create free database
  • Get host, port, and password
Go to Redis
⚠️ Save Your Credentials! You'll need these in Step 4.

📤 Step 3: Deploy to Vercel

Option A: Deploy with GitHub (Recommended)

  1. Create GitHub repository
  2. Push your code to GitHub
  3. Go to vercel.com
  4. Sign in with GitHub
  5. Click "Add New Project"
  6. Import your repository
  7. Click "Deploy"
git init
git add .
git commit -m "Initial commit"
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
Go to Vercel Create GitHub Repo

Option B: Deploy with Vercel CLI

npm install -g vercel
vercel
