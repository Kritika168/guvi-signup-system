Set up your free cloud databases in 3 steps

Step 1: MySQL Database (PlanetScale)

📦 What is it? PlanetScale provides free MySQL databases. We'll use it for user authentication (username, email, password).
Instructions:
  1. Click the button below to open PlanetScale
  2. Sign up with your GitHub account (free)
  3. Click "Create a new database"
  4. Name it: guvi_user_db
  5. Select region: AWS - Mumbai (ap-south-1) (closest to India)
  6. Click "Create database"
  7. Click "Connect" → Select "PHP/MySQL" → Copy the credentials
  8. Find: host, username, password, database name and paste below
  9. IMPORTANT: Go to "Console" tab and run this SQL:
    CREATE TABLE users ( id INT PRIMARY KEY AUTO_INCREMENT, username VARCHAR(50) UNIQUE NOT NULL, email VARCHAR(100) UNIQUE NOT NULL, password VARCHAR(255) NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP );

Step 2: MongoDB Database (MongoDB Atlas)

📦 What is it? MongoDB Atlas provides free NoSQL databases. We'll use it for user profiles (name, age, contact, address).
Instructions:
  1. Click the button below to open MongoDB Atlas
  2. Sign up with your email (free)
  3. Click "Build a Database"
  4. IMPORTANT: Select "M0 FREE" tier (NOT M2/M5/M10!)
  5. Provider: AWS, Region: Mumbai (ap-south-1)
  6. Cluster Name: GuviCluster
  7. Create username: guviuser and a strong password
  8. Add IP: 0.0.0.0/0 (allow all) - for development
  9. Click "Connect" → "Connect your application" → Copy the connection string
  10. Replace <password> in connection string with your actual password
  11. Replace test with guvi_profiles in the connection string
Enter Your Connection String:
Example: mongodb+srv://guviuser:mypass123@guvicluster.abc12.mongodb.net/guvi_profiles
✓ MongoDB Configured

Step 3: Redis Database (Redis Cloud)

📦 What is it? Redis Cloud provides free in-memory databases. We'll use it for session management (login sessions).
Instructions:
  1. Click the button below to open Redis Cloud
  2. Sign up with your email (free)
  3. Click "New Subscription" → Select "Free" plan (30MB)
  4. Provider: AWS, Region: ap-south-1 (Mumbai)
  5. Click "Create database" → Database name: guvi_sessions
  6. After creation, click on database → Copy:
    • Endpoint (e.g., redis-12345.c123.ap-south-1-1.ec2.cloud.redislabs.com:12345)
    • Default user password
  7. Split endpoint into host and port (before and after the colon)

💡 Need Help?
  • MongoDB says "not free": Make sure you select M0/Shared tier, NOT M2 or higher
  • Can't find credentials: Look for "Connect" or "Connection String" buttons in each service
  • Server not working: Make sure to restart the server after updating .env file
  • Signup fails: Check if you ran the SQL CREATE TABLE command in PlanetScale Console
