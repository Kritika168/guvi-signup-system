1

Sign Up for InfinityFree Hosting

InfinityFree provides free PHP hosting with MySQL database.

  • Free PHP & MySQL hosting
  • No ads on your site
  • Free subdomain (yourname.infinityfreeapp.com)
Sign Up Now
2

Sign Up for MongoDB Atlas

MongoDB Atlas provides free cloud MongoDB database (512MB).

  • Create account and sign in
  • Create a FREE M0 cluster (select AWS/GCP/Azure - any region)
  • Setup Database User (username & password)
  • Setup Network Access: Allow 0.0.0.0/0 (Allow from anywhere)
  • Get connection string: Click "Connect" → "Connect your application"
Save your MongoDB connection string:
Sign Up for MongoDB
3

Sign Up for Redis Cloud

Redis Cloud provides free Redis database (30MB).

  • Create account and sign in
  • Click "New Database" and select FREE plan
  • Choose region closest to your hosting
  • Note down: Endpoint, Port, and Password
Save your Redis credentials:
Sign Up for Redis
4

Upload Files to InfinityFree

Upload all project files to your hosting account.

  • Login to InfinityFree Control Panel (VistaPanel)
  • Go to "Online File Manager"
  • Navigate to htdocs folder
  • Upload ALL files from your deploy_package folder
  • Or use FTP (FileZilla) for faster upload
5

Setup MySQL Database

Create MySQL database and import schema.

  • In control panel, go to "MySQL Databases"
  • Click "Create Database" and create new database
  • Note down: Database name, Username, Password, Hostname
  • Go to phpMyAdmin
  • Select your database and click "Import"
  • Upload sql/schema.sql file
Save your MySQL credentials:
6

Update Configuration Files

Update database credentials in your configuration files.

  • Edit php/config/mysql_config.php with MySQL credentials from Step 5
  • Edit php/config/mongodb_config.php with MongoDB URI from Step 2
  • Edit php/config/redis_config.php with Redis credentials from Step 3
